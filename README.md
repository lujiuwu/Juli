# **“橘颂荣邑”助农小程序**
`uni-app`  `u-view`

## 项目简介
“橘颂荣邑”助农小程序是一个线上助农销售系统，主要功能为帮助园果农处理农产品与各类农副产品，基于uni-app技术并依托微信小程序平台开发
